      How the ERPNext integration works

      Counts are pushed as a draft Stock Reconciliation (Stock module — the same doctype ERPNext uses for physical inventory counts and POS warehouse reconciliation). Review and submit the draft inside ERPNext to post the adjustment. Create the API key/secret under User → API Access for a user with Stock Manager role, and allow CORS for this app's origin (allow_cors in site_config.json).
